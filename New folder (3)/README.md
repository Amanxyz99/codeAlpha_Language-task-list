# Language Translation Tool

A simple web-based language translation tool that allows users to translate text between multiple languages using the MyMemory Translation API.

## Features

- Translate text from one language to another
- Support for multiple languages including English, Hindi, French, Spanish, German, Italian, Portuguese, Chinese, Japanese, Russian, Arabic, Korean, Dutch, and Swedish
- Auto-detect source language option
- Copy translated text to clipboard
- Responsive design for mobile and desktop use
- Error handling for failed translations

## Technologies Used

- HTML
- MyMemory Translation API

## How to Use

1. Open `index.html` in a web browser
2. Enter the text you want to translate in the textarea
3. Select the source language (or choose "Auto Detect")
4. Select the target language
5. Click the "Translate" button
6. View the translated text in the output area
7. Optionally, click "Copy Translation" to copy the result to your clipboard

## API Usage

This tool uses the free MyMemory Translation API (https://mymemory.translated.net/). Please note that the API has usage limits for free accounts.

## Browser Compatibility

Works in all modern web browsers that support ES6 JavaScript and the Fetch API.
